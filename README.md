# Five Crowns Helper

A web application to help track scores and find melds for the Five Crowns card game.

## Features

- **Scoresheet**: Track scores across all 11 rounds with automatic total calculation
- **Meld Helper**: Find optimal meld combinations and get discard suggestions
- **Play Game**: Pass-and-play game mode with full game state management

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Heroicons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This project is configured for Vercel deployment. Simply connect your repository to Vercel and it will automatically build and deploy.

### Vercel Configuration

- Framework: Vite
- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

The project includes a `vercel.json` file that handles SPA routing.
